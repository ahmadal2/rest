# Footer Editing Guide for Admins

## Overview
This guide explains how administrators can edit the footer content of the website. The footer is now fully editable for all authorized admins (main admin and secondary admins 1, 2, 3).

## Prerequisites
- Valid admin credentials
- Admin role: Main Admin or Secondary Admin with number 1, 2, or 3
- Edit mode enabled

## How to Edit Footer Content

### 1. Enable Edit Mode
1. Log in to the admin panel
2. Click the "Edit Mode" toggle button in the navigation bar
3. The edit mode overlay will appear at the top of the screen

### 2. Navigate to Footer
1. Scroll to the bottom of the page to view the footer
2. Hover over any editable text element
3. A small edit icon will appear in the top-right corner of the element

### 3. Edit Content
1. Click on any text element to open the edit dialog
2. Modify the text in the input field
3. Use the following controls:
   - **Save**: Apply changes (Enter key or Ctrl+Enter for multiline)
   - **Cancel**: Discard changes (Escape key)

### 4. Editable Footer Elements
The following footer elements can be edited:

#### Brand Section
- **Site Name**: The main brand name displayed in the footer
- **Footer Description**: The descriptive text about the restaurant

#### Quick Links Section
- **Quick Links Title**: The heading for the quick links section
- **Link Names**: Names of navigation links (About, Menu, Reservations, Contact)

#### Contact Information Section
- **Contact Title**: The heading for the contact section
- **Address**: Physical address of the restaurant
- **Phone**: Contact phone number
- **Email**: Contact email address

#### Newsletter Section
- **Newsletter Title**: The heading for the newsletter section
- **Newsletter Description**: Descriptive text for the newsletter
- **Subscribe Button**: Text on the subscribe button

#### Bottom Bar
- **Copyright Text**: Copyright and rights reserved text
- **Made With Text**: "Made with" text
- **Legal Links**: Privacy Policy, Terms of Service, Cookie Policy

## Saving Changes
All changes are automatically saved to localStorage and will persist between sessions. Changes are immediately visible to all visitors.

## Permissions
- **Main Admin**: Full editing access to all footer elements
- **Secondary Admin #1-3**: Full editing access to all footer elements
- **Secondary Admin #4+**: No footer editing access (dessert management only)

## Best Practices
1. Keep text concise and clear
2. Maintain consistent branding
3. Ensure contact information is accurate
4. Test links after editing
5. Use appropriate language for each translation

## Troubleshooting
- If edit mode doesn't activate, ensure you're logged in as an authorized admin
- If changes don't save, check browser console for errors
- If text doesn't update, refresh the page to see cached changes
